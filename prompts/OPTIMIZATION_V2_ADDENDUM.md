## 🎯 P0-1: Edition Master Summary (SSOT) - 所有文章必須遵守

### 關鍵數字口徑統一
所有三篇文章必須使用完全一致的數字口徑：
- **資料期間**: 明確標註 TTM / Q3'25 / 2026-01-19 close
- **口徑**: GAAP vs Non-GAAP 必須一致
- **來源**: 所有數字必須標註來源（FMP / 10-Q / fact_pack）

### 統一情境假設
Base/Bull/Bear 情境的定義必須在三篇中保持一致：
- 🔴 Bear case: 觸發條件、價格區間、關鍵指標門檻
- 🔵 Base case: 觸發條件、價格區間、關鍵指標門檻  
- 🟢 Bull case: 觸發條件、價格區間、關鍵指標門檻

**示例**:
```
Base Case: IREN $50-65, US10Y 4-4.5%, FCF改善 >20% QoQ
Bull Case: IREN $75-90, US10Y <4%, FCF轉正
Bear Case: IREN $35-40, US10Y >5%, FCF持續惡化
```
